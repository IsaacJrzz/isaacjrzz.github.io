# 🚀 Isaac Alonso | Software Developer Portfolio

¡Bienvenido a mi portfolio personal! Este proyecto es una **Single Page Application (SPA)** diseñada para mostrar mi trayectoria, proyectos y habilidades técnicas de una manera directa, rápida y visualmente atractiva.

---

## ⚡ Tecnologías Utilizadas

Este proyecto ha sido desarrollado utilizando las siguientes tecnologías:

* **React 18** (Vite como bundler por su velocidad extrema)
* **Tailwind CSS** (Para un diseño moderno y responsive)
* **Framer Motion** (Animaciones e interactividad suave)
* **Lucide React** (Iconografía minimalista)
* **React Scroll** (Navegación fluida entre secciones)

---

## 📂 Estructura del Proyecto "All-in-One"

El portafolio se divide en las siguientes secciones integradas en una única vista:

* **Hero:** Introducción impactante y propuesta de valor.
* **About Me:** Mi historia y stack tecnológico.
* **Projects:** Galería de trabajos destacados con enlaces a demos y repositorios.
* **Skills:** Mis herramientas principales (Frontend, Backend, Tools).
* **Contact:** Formulario y enlaces directos a mis redes profesionales.
